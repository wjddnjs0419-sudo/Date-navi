# Step Intent Phase 4 — 연기 기록 (가격/외부증거)

> 결정: 2026-07-19 · 사용자 확정("연기 — 문서화만")
> 관련: `docs/superpowers/specs/2026-07-19-step-intent-phase2-3-design.md` §7

## 무엇이었나

Phase 4는 스텝 intent 매칭에 **가격 적합성·외부증거(평점·리뷰수)** 소프트 시그널을 더해 랭킹 품질을 높이려는 단계였다.

## 왜 연기하나 (블로커)

- 현재 파이프라인의 **유일한 장소 데이터 소스인 카카오 로컬 검색 무료 티어는 가격·평점을 반환하지 않는다.** 외부증거 시그널을 계산할 원천 데이터가 없다.
- `recommendation-prompt.ts`의 price 언급은 프롬프트 텍스트일 뿐 실데이터가 아니다(스펙 §17 확인).

## 착수 선행 조건 (별도 세션)

1. **데이터 소스 확보** 중 택1:
   - 네이버/구글 플레이스 등 **유료 API 배선** — 비용·키 발급·쿼터, 크로스유저 캐시 프라이버시 재검토 필요.
   - 자체 크롤링 — ToS/법률 리스크 검토 필요.
2. 확보 후: evidence 스키마에 가격대/평점 필드 추가 → 랭킹 가중치(`RANKING_SCORE_WEIGHTS`)에 가격 적합·평점 슬롯 신설 → TDD.

## 현 세션 산출물

- 코드 변경 없음. 본 문서로 블로커·선행조건만 기록.
- Phase 2(서버+AI)·Phase 3(완화 UI)은 완료. Phase 4만 데이터 게이팅으로 대기.
